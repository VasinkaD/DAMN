# From Stars to Molecules: AI Guided Device-Agnostic Super-Resolution Imaging

This repository provides data and supplementary material for the paper **From Stars to Molecules: AI Guided Device-Agnostic Super-Resolution Imaging**, by Dominik Vašinka, Filip Juráň, Jaromír Běhal, and Miroslav Ježek. <br>
The paper is available on [arXiv](https://doi.org/10.48550/arXiv.2502.18637). The repository has been updated for paper v3, which is to be uploaded to arXiv soon.

In addition to this GitHub repository, the DAMN ConvNet and ResNet models, along with the experimental dataset, are stored in the [Zenodo repository](https://doi.org/10.5281/zenodo.14641651) due to their size.

## The repository structure:
### Demos
This directory contains code and data for introduction and basic demos on running the DAMN model.

### Device_agnostic_training
This directory contains code for training a device-agnostic model using incremental learning with simulated data.

### Paper_results
This directory contains all the code and data to reproduce the results presented in the paper.
