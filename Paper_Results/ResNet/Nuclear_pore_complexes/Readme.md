This directory contains code and data to reproduce the results for the nuclear pore complexes dataset from the section "*Astronomical and localization microscopy demonstration*" of the main text 
and *Figure 6-II*. <br>
In addition to the provided files, it also requires the *DAMN_ResNet.keras* model stored in the *ResNet* directory. <br>

-> **Process_NPC.ipynb** <br>
A Jupyter notebook that processes the nuclear pore complexes dataset and visualizes its results. <br>

-> **func_file_Model.py** <br>
Architecture definitions for creating the DAMN ResNet model and subsequently loading the weights. <br>

-> **func_file_NPC.py** <br>
Supporting functions for processing data. <br>

-> **DS_single_NPC.npy** <br>
The Deep-STORM reconstruction of a single NPC using 5,000 low-resolution frames, as. <br>
